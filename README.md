# Fake News Detection using NLP

This project detects whether a news article is **Fake** or **Real** using
Natural Language Processing (NLP) techniques and Machine Learning.

---

## Project Overview
- Text data is converted into numerical features using **TF-IDF**
- A **Logistic Regression** model is trained to classify news
- The trained model and vectorizer are saved for reuse

---

## Model Details
- Vectorizer: TF-IDF
- Algorithm: Logistic Regression
